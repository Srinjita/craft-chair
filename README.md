# craft-chair

Parameterized chair model.

### Install
	$ npm install craft-chair

### Parameters
- size: scales model
- leg_height: adjusts height of chair legs
- length: adjusts length of seat

### Example
```html
<craft>
	<craft name="chair" module="craft-chair"/>
	<chair size="2"></chair>
	<chair></chair>
</craft>
```
